引用与指针

什么是引用 
int ival = 1024;
int &refVal = ival; //refVal指向ival(是ival的另一个名字)
int &refVal2;       //报错：引用必须被初始化

引用即别名，一旦初始化完成，引用将和它的初始值对象一直绑定在一起

引用与指针的区别
  1. 引用必须初始化，而指针不需要
  2. 指针可以在生命周期内先后指向几个不同的对象，而引用只能与初始值对象绑定在一起
能否定义指向引用的指针
不能，因为引用不是对象，没有实际地址，所以不能定义指向引用的指针
